# paygoal-excercise
 Excercise for the position of Developer at Paygoal

Documentation for the API  is available in file API-Documentation

## Requierements

- Java 11
- Maven

### How to run the project.
```bash
mvn clean install
```
```bash
java -jar app/target/app-1.0-SNAPSHOT.jar
```
